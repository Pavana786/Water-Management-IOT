# Underpass Water Management System

## Overview

An IoT-based smart water management system designed to monitor water accumulation in underpasses and help prevent flooding.

## Features

* Water level monitoring
* Real-time alerts
* Flood prevention support
* Smart infrastructure monitoring

## Technologies Used

* IoT
* Embedded Systems
* Sensors
* Wireless Communication
